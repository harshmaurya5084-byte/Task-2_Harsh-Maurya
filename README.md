# Harsh Maurya — Personal Portfolio 
# Decodes Task 2

A modern, responsive portfolio website showcasing projects, skills, and achievements.

## Tech Stack
- **HTML5** — Semantic markup with accessibility attributes
- **CSS3** — Glassmorphism, gradients, animations, media queries
- **JavaScript** — Typing effect, scroll-reveal, active nav highlighting

## Design
- **Theme:** Dark navy (`#0a0e1a`) with glassmorphism cards
- **Colors:** Neon cyan (`#00f0ff`), electric purple (`#7c3aed`), hot pink (`#f472b6`), emerald (`#34d399`)
- **Effects:** Frosted glass cards, neon glow shadows, gradient borders, ripple buttons
- **Typography:** Fluid sizing using `clamp()` across all headings and body text

## Features
- Responsive across all screen sizes (desktop → mobile → 360px)
- CSS-only hamburger menu with animated toggle
- Typing animation cycling through role titles
- Scroll-reveal animations via `IntersectionObserver`
- Floating gradient orbs in the hero section
- Scroll-to-top button with glow effect
- `prefers-reduced-motion` accessibility support
